# 🌐 Task Manager Frontend (React)

This is the frontend of the Task Manager application built using React.js. It provides a user-friendly interface to manage tasks with authentication and real-time updates.

---

## 🚀 Live Demo
cutetask.netlify.app



---

## 🛠️ Tech Stack

- React.js
- Axios
- CSS

---

## ✨ Features

- 🔐 User Login & Registration
- 📋 View Tasks
- ➕ Add New Tasks
- ✏️ Edit Tasks
- ❌ Delete Tasks
- 🔄 API Integration with Backend
